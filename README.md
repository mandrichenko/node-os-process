# Node.js: OS process

Enables you to search, kill and list local system processes.

## Support

Currently supports only Windows.

## Usage

```javascript
var ps = require('node-os-process');
```

## Functions

**findByName(processName, callback)**

**list(callback)**

**kill(pid, force, callback)**
